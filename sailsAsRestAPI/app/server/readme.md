# REST API

> Basic Rest API built with [SailsJS](http://sailsjs.org/)

### Features

- Rest API (server App)
- User authentication and session control (JWT, register, login, logout)

## Dev

install the server app dependencias

```
$ npm install
```

### Run

```
$ npm start
```

### Notes

generated with:

```
$ sails generate backend
```
